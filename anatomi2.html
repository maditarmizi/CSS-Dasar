1. Anatomi CSS

Anatomi CSS Adalah Susunan/Struktur yang digunakan untuk menulis syntaks CSS Dengan benar. Contoh Susunan CSS Di bawah ini

selector { properti: value; }

h1 { color: blue; }

Cara Mendeklarasikan : CSS, tolong pilih elemen h1, Lalu ubah teks didalamnya menjadi berwarna biru Ket:
h1 = Sebagai Selector
color = Sebagai Properti
blue = Sebagai Value/Nilai

2. Pengertian Selector

  • Digunakan untuk memilih dan memanipulasi elemen spesifik pada HTML
  • Elemen HTML dipilih berdasarkan tag,id,class bahkan pola/pattern
  • Semakin kompleks struktur HTML, Selector juga bisa semakin kompleks/spesifik
