Text Styling

1. Color

memberi warna pada tulisan

warna : red,lightsegreen,royalblue
hexadesimal : #ff0000,#20b2aa,#4169e1
rgb/red green blue : rgb(255,0,0), rgb(22,178,172), rgb(65,105,225)

klik

2. text-align

mengatur format paragraf/text : left/kiri,right/kanan,center/tengah,justify/kiri-kanan

3. text-indent

memberi indentasi pada paragraf/text : px,%

4. text-decoration

mengatur decoration pada text : none,underline,overline,line-through

5. text-transform

mengubah jenis huruf menjadi huruf besar,kecil/kapital : none,lowercase,uppercase,capitalize

7. letter-spacing

mengatur spasi antar huruf : normal,px

word-spacing

mengatur spasi antar kata : normal,px
