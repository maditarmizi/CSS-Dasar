Jenis Font

Materi CSS Tentang Font Styling

1. font-family

• mengatur jenis font yang akan digunakan

Contoh : 1. nama font 2. generic family

body { font-family: arial, verdana, sans-serif; }
p { font-family: "Times New Roman", serif; } Ket: nama font: arial,verdana
generic family: sans-serif
• sans-serif adalah font" yang tidak memilki kaki
• serif adalah font" yang memilki kaki

2. font-size

• mengatur ukuran font : px,%,em

3. font-weight

• mengatur ketebalan font : lighter,normal,100-900,bold,bolder

4. font-variant

• mengubah font menjadi small caps : normal, small-caps

5. font-style

• mengubah font menjadi bercetak miring : normal,italic,oblique

6. line-height

• mengatur spasi antar baris : normal,px,em
