Pengertian Selector

digunakan pada CSS untuk mengenali sebuah elemen HTML yang akan diberi style. untuk lebih jelasnya, silahkan kunjungi website ini Klik. Selector: elemen HTML,id,class,complex selector

ID

• sebuah elemen html hanya boleh memilki 1 id

• setiap halaman hanya boleh memilki 1 elemen dengan id tersebut

• dapat digunakan sebagai penanda halaman untuk link

• digunakan juga untuk javascript

• sebaiknya tidak digunakan untuk CSS(lebih baik gunakan class)
