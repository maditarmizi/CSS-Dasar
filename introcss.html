1. Pengertian CSS

CSS Adalah Mekanisme Sederhana yang mengatur gaya / style. Contoh : Warna,Ukuran,Posisi dll. Pada Suatu Website

Menurut Source diatas : Klik Halaman

2. Teori CSS

  • Aturan yang digunakan untuk menampilkan elemen / tag HTML
  • Dibuat terpisah dengan HTML
  • Bertujuan untuk memisahkan konten & style
  • 1 CSS dapat digunakan untuk banyak halaman HTML
  • 1 halaman HTML dapat terlihat berbeda jika menggunakan CSS yang berbeda pula Klik
